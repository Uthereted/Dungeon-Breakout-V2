# TEFG Project

## Overview
The TEFG project focuses on procedural generation techniques applied to dungeon design and gameplay mechanics. The main objective is to create random dungeon layouts, enemy spawns, and item placements that offer a unique experience for players every session.

## Features
- **Procedural Dungeon Generation**: Automatically generates dungeon layouts based on predefined algorithms, ensuring no two runs are the same.
- **Dynamic Enemy Spawns**: Enemies are spawned dynamically based on the player's location, previous actions, and the current dungeon state.
- **Randomized Item Placement**: Items are placed randomly throughout the dungeon, enhancing exploration and discovery.

## Goals
- To create an engaging player experience through unpredictable environments.
- To implement efficient algorithms that minimize loading times while maximizing replayability.

## Technologies Used
- Unity Engine
- C#
- Perlin Noise for random generation

## Conclusion
The TEFG project aims to revolutionize how dungeons are created in games, offering a fresh perspective on procedural generation and player engagement.
